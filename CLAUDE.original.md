# ExpensesTrackerAI

## What Is This

ExpensesTrackerAI is a beautifully simple, natively iOS personal finance app for couples. 
Inspired by Apple's Health app, it makes tracking spending feel natural and insightful — Simple, Smart, Clear, and unmistakably Apple.

## Code organization

- Views: UI only — no logic/data transform
- ViewModels: business logic, filtering, calculations, state management
- `Core/Foundation/` extensions: reusable utilities shared across the app
